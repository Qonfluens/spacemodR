# SF object defining very simplified OCS-GE soil cover metaleurop

Simple feature collection with 9 features and 11 fields. Projected CRS:
RGF93 v1 / Lambert-93.

## Usage

``` r
data(ocsge_metaleurop)
```

## Format

An object of class `sf` (inherits from `data.frame`) with 9 rows and 12
columns.

## Examples

``` r
data(ocsge_metaleurop)
```
