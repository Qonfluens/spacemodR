# Add non-habitat zones to a Habitat object

Add non-habitat zones to a Habitat object

## Usage

``` r
add_nohabitat(hab, sf_data, ...)
```

## Arguments

- hab:

  A Habitat object

- sf_data:

  An sf object containing geometries to add

- ...:

  Additional arguments

## Value

A Habitat object with new geometries added as non-habitat
